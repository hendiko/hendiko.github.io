jQuery-Cookie

jquery-cookie

基本用法

创建 cookie：

$.cookie('name', 'value');

创建具有有效期的 cookie：

$.cookie('name', 'value', {expires: 7});

创建具有有效期的 cookie，全站有效：

$.cookie('name', 'value', {expires: 7, path: '/'});

读取 cookie：

$.cookie('name');

读取全部 cookie：

$.cookie();

删除 cookie：

$.removeCookie('name');

// Need to use the same attributes (path, domain) as what the cookie was written with
